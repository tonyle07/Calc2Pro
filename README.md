# Calc2Pro

A Calculus II study platform built with React and deployed on Vercel.

**Live site:** https://calc2pro.vercel.app

## Features
- 25 units of study notes with key concepts and formulas
- AI-powered tutoring via the Anthropic Claude API
- Custom quiz builder with difficulty levels and timed mode
- Firebase Authentication (Google + email)
- Firestore cloud sync for progress tracking

## Tech Stack
React, Vite, Firebase, Anthropic Claude API, Vercel
